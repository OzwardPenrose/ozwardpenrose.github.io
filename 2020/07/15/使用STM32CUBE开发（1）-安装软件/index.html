STM32程序开发的捷径

随着STM32单片机的发展，功能和外设的不断增加，开发难度不断提高，寄存器、库函数的开发方式已经渐渐不能适用于现今的要求快捷的开发，我们越来越需要全新的开发方式来解放我们的繁琐劳动，意法半导体就在此时推出了STM32CUBE开发工具。

这是一个轻量级的开发工具，它的大小只有百余兆，只需要搭配足够的软件HAL库，它几乎可以应用于所有M4架构的芯片——这一成果无疑大大减少了嵌入式工程师的工作，可谓是STM32单片机开发的捷径。

我们先学习如何安装这个软件。

安装JRE

JRE（Java Runtime Environment）——Java运行环境。STM32CUBE是基于Java环境运行的，所需需要安装。

官网地址：https://www.java.com/en/download/manual.jsp

根据自己的系统需要选择合适的jre，之后点击exe文件无脑安装即可。

安装STM32CubeMX

下载地址：STM32Cube官网

下载完成。

等待安装完成即可。

安装完成点击done退出即可。

安装HAL库

STM32 HAL固件库是Hardware Abstraction Layer的缩写，中文名称是：硬件抽象层。HAL库是ST公司为STM32的MCU最新推出的抽象层嵌入式软件，为更方便的实现跨STM32产品的最大可移植性。HAL库的推出，可以说ST也慢慢的抛弃了原来的标准固件库，这也使得很多老用户不满。但是HAL库推出的同时，也加入了很多第三方的中间件，有RTOS，USB，TCP / IP和图形等等。

HAL库能实现STM32系列单片机代码之间的无缝切换，并且能够快速移植。

从2016年开始ST公司开始停止了对固件库的更新，并且加大了使用HAL库开发的力度，可以预见这会成为后续嵌入式开发的主流方式。

HAL库可以采用在线安装和离线安装两种方式。

在线安装

打开STM32cube界面，点击help->manage embedded software packages。

根据需要选择对应的芯片内核还有版本号。

离线安装

官网下载地址：stm32cube

将这个下载包解压后放到对应的文件夹下面即可。
